DELETE THESE IF YOU SEE THEM
  manifest.json and "manifest (1).json" were auto generated by PWABuilder
  from the GitHub repository page. They describe GitHub, not this app, and
  will confuse the install. Delete them from the folder and the repository.

FINISHING THE GITHUB SETUP (your repository: georgefournier111/Life-168)
  1. Upload every file listed above to the top level of the repository
  2. Delete manifest.json and "manifest (1).json" from the repository
  3. In the repository, click Settings
  4. Click Pages in the left sidebar
  5. Source: "Deploy from a branch"
  6. Branch: main   Folder: / (root)   then click Save
  7. Wait about a minute. The address will be:
       https://georgefournier111.github.io/Life-168/
  8. Confirm it worked: visit
       https://georgefournier111.github.io/Life-168/sw.js
     You should see the service worker code, not a "404" page.

HOW PEOPLE INSTALL IT AS AN APP
  iPhone or iPad: open the address in Safari, tap Share, "Add to Home Screen"
  Android: open the address in Chrome, tap the three dot menu,
    "Add to Home screen"
  It opens full screen with the 168 icon, works offline, and each person's
  answers stay saved on their own device.

TO UPDATE THE APP LATER
  Replace index.html with the new version and change the cache name in
  sw.js from "wlll168-v4" to "wlll168-v5" (then v6, and so on) so
  installed copies pick up the update.

THEN FOR GOOGLE PLAY
  Once https://georgefournier111.github.io/Life-168/ is live, go to
  https://www.pwabuilder.com and enter THAT address (not the github.com
  repository address). It will find the service worker and generate the
  Android package for the Google Play Console ($25 one time).
